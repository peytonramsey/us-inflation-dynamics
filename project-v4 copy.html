ADF Testing

Testing for stationarity in our plots, will help us determine which model is best to use (VAR or VECM). If it’s stationary, it’d be best to use VAR, otherwise if there is non-stationarity we will use run a Johansen test, if r = 0, there’s no cointegration and we can use VAR, if r >= 1, there is cointegration and we will use VECM.

ts_df_diff <- df[, -1]

adf_test_diff <- function(x) {
  differenced_x <- diff(x, differences = 1)
  test_result <- adf.test(differenced_x, alternative = "stationary")
  return(test_result$p.value)
}

However, ind_pro.dl1 shows no significance in predicting CPI in the short run, though unemp_rate.dl1 is significant in the ind_pro.d and unemp_rate.d equations, suggesting unemployment and industrial production are more closely linked to each other than to inflation directly.
